# Physics-Based Design and CAD Modelling of a Coaxial Non-Premixed Burner

This repository contains the Python implementation of a physics-based design tool for a coaxial non-premixed burner. The tool performs analytical burner sizing, design verification, automatic nozzle profile generation using fifth-order polynomial curves, and exports CAD-ready coordinate files for SolidWorks geometry construction.

📄 **Project Report:** `Burner_Design.pdf`

The report provides a detailed explanation of:
- Burner design methodology
- Combustion calculations
- Fuel tube and casing sizing
- Design verification
- Nozzle profile generation
- Mathematical formulation
- Python implementation
- Results and discussion
